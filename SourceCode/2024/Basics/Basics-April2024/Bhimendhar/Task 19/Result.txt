Productivity Activities
      
      
      
         
         
            Overviewlink
            Activities included in the Productivity category allow you to
               build basic automation projects using emails, spreadsheets, file storing and sharing
               services, Word files, and presentations. 
            
            
            Some of the most common use cases are:
               
                  
                  Managing CSV and Excel files.
                  
                  
                  Manipulating data tables in
                     spreadsheets.
                  
                  
                  
                  Managing emails.
                  
                  
                  Working with PowerPoint
                     presentations.
                  
                  
                  
                  Working with Microsoft Word files.
                  
                  
                  Triggering specific actions in the
                     workflow according to specific events.
                  
                  
                  
               
               
            
            
            
            
               PackageDescriptionExcel
                           The Excel activities package contains activities that enable you to read
                              information from a cell, columns, rows or ranges, write to other spreadsheets or
                              workbooks, execute macros, and even extract formulas. 
                           
                           
                           To learn more, go to
                              the Excel Activities Package section. 
                           
                           
                        MailThe Mail Activities package is designed to facilitate the automation of
                           mail-related tasks. 
                           To learn more, go to the Mail Activities Package section.
                           
                           
                        Google WorkspaceThe Google Workspace activities package helps you automate Google Workspace
                           (formerly known as G Suite) applications, including Google Calendar, Google Drive,
                           Google Sheets, Gmail, and Google Docs. 
                           To learn more, go to the Google Workspace
                              Activities Package section. 
                           
                           
                        Microsoft 365The Microsoft 365 Activities Package gives you the ability to automate
                           interactions with Microsoft 365 applications: OneDrive & SharePoint, Excel,
                           Calendar, and Outlook. 
                           To learn more, go to the Microsoft 365 package section.
                           
                           
                        PresentationsThe Presentations Activities Package helps you automate tasks you perform in
                           presentations using Microsoft PowerPoint.
                           To learn more, go to the Presentations
                              Activities Package section.
                           
                           
                        WordThe Word package contains multiple activities that enable you to manipulate
                           .docx files. You can use the activities to add images to your
                           documents, read, append or replace text within them, add bookmarks, as well as
                           export them to PDF.
                           To learn more, go to the Word Activities Package section.
